深度学习推荐系统实战之Bandit

深度学习推荐系统实战
推荐算法
发布日期:   2021-04-02

Bandit起源

Bandit算法来源于历史悠久的赌博学。想象这样一个场景：一个赌徒走进赌场，一眼看去是一排外表一摸一样的老虎机。每个老虎机吐钱的概率不一样，他应该在每次如何选择老虎机才可以做到最大化收益？这就是多臂赌博机问题（Muti-arm Bandits，简称MAB）。一系列应对该问题的策略统称为Bandit算法。

现实生活中我们也会遇到一系列类似的场景：

  1. 新用户进入系统，如何得知他对哪个类别更感兴趣？
  2. 有若干广告库物料，如何展示广告，才能获得最大收益？是一直展示收益最高的吗？
  3. 玩家进入新游戏，哪个关卡是他最感兴趣的，如何合理排序？

解决这类问题最好的方法就是去试，但也不是随便试，而是有策略的试，通过多次实验摸透每个老虎机背后的赢钱规律（概率分布）。

相关概念

累计遗憾

Bandit算法的思想是，看看每次选择会带来多少遗憾，累计遗憾越少越好。

$R_{T}=\sum_{i=1}^{T}(w_{opt}-w_{B(i)})=Tw^{*} - \sum_{i=1}^{T}w_{B(i)}$

$w_{opt}$是每次最优选择得到的收益，$w_{B(i)}$是每次实际选择得到的收益，二者的差就是遗憾的量化。进行T次选择之后，就获得累计遗憾$R_{T}$。

Bandit算法

为了更好的学习Bandit算法，我们可以带入下场景。一名拉斯维加斯赌徒走进了赌城，面前有5台老虎机，每台老虎机上都有一个拉杆，如何通过有限的尝试知道哪台老虎机赢钱概率最大？下面介绍多种应对的策略。

朴素bandit

随机试若干次，计算每个臂的平均收益，一直选均值最大的那个臂。这个算法是在实际生活中最常采用的。

Epsilon-Greedy

Epsilon贪婪算法原理

  1. 在(0, 1)之间选择一个数，名为epsilon。
  2. 每次以epsilon概率随机选择一个臂，以1-epsilon概率选择平均收益最大的臂。
UCB

置信区间上界（Upper Confidence Bound，UCB），为每个臂评分，选择评分最高的候选臂。然后观察用户反馈，更新候选臂。

$x_{j}^{-}(t) + \sqrt{\frac{2lnt}{T_{j,t}}}$

t是目前的总选择次数，$x_{j}^{-}$是每个臂的平均收益，$T_{j,t}$是j臂的选择次数。由公式可知，一个候选臂被选择次数$T_{j,t}$越小，它的价值越大，排序越靠前。（从公式上可以看出，除了臂本身的平均收益之外，增加了一个选择次数的纬度，相同次数下，选择次数越少，值越大。）

Thompson Sample

汤普森采样算法原理

  1. 假设每个臂背后的概率分布是Beta分布。
  2. 每次选择，每个臂都会出一个随机数，然后按随机数排序，输出最大随机数的臂对应的物品。

代入到推荐场景中。Beta分布有两个参数，a参数代表用户点击次数，b参数代表用户没有点击次数，然后进行汤普森采样。

  1. 取出每一个候选臂对应的参数a和b。
  2. 每个候选臂根据Beta(a,b)产生一个随机数。
  3. 按随机数排序，输出最大值对应的候选臂。
  4. 根据用户反馈，用户点击则a+1，否则b+1。
import math
import numpy as np
import matplotlib.pyplot as plt

class Bandits:
    
    def __init__(self):
        self.n_bandits = 10 # 10组老虎机
        self.n_arms = 5 # 5台老虎机（每台老虎机1个臂）
        self.n_pulls = 1000 # 总共拉1000次
        self.strategies = ['random', 'epsilon_greedy', 'ucb', 'thompson_sample']
        
        # epsilon_greedy参数
        self.epsilon = 0.3
        self.decay_rate = 0.999
        self.min_epsilon = 0.1
        
    def pick_arm(self, strategy, q_values, counts, success, failure):
        """ 选择臂
        :strategy 策略
        :q_value 每个臂的收益
        :counts 总次数
        :success 收益
        :failure 遗憾
        """
        if strategy == 'random':
            return np.random.randint(0, len(q_values))
        
        elif strategy == 'epsilon_greedy':
            # [0.1, 0.3]
            epsilon = max(self.epsilon*self.decay_rate, self.min_epsilon)
            if np.random.random() > epsilon:
                best_arms_value = np.max(q_values)
                best_arms = np.argwhere(q_values == best_arms_value).flatten()
                return best_arms[np.random.randint(0, len(best_arms))]
            else:
                return np.random.randint(0, len(q_values))
            
        elif strategy == 'ucb':
            total_counts = np.sum(counts)
            ucb_q_values = q_values + np.sqrt(np.reciprocal(counts+0.001)*2*math.log(total_counts+1.0))
            best_arms_value = np.max(ucb_q_values)
            best_arms = np.argwhere(ucb_q_values == best_arms_value).flatten()
            return best_arms[np.random.randint(0, len(best_arms))]
        
        elif strategy == 'thompson_sample':
            """ 利用beta分布选择杆 """
            sample_means = np.zeros(len(counts))
            for i in range(len(counts)):
                sample_means[i] = np.random.beta(success[i]+1, failure[i]+1)
            return np.argmax(sample_means)
            
    def plot_result(self):
        """ 绘制曲线 """
        fig = plt.figure()
        ax = fig.add_subplot(111)
        
        for st in self.strategies:
            # 初始化
            # 记录10x5台老虎机拉杆次数
            best_arm_counts = np.zeros((self.n_bandits, self.n_pulls))
            for i in range(self.n_bandits):
                # 每个杆是最大收益的概率
                arm_prob = np.random.rand(self.n_arms)
                # 收益最大的杆
                best_arm = np.argmax(arm_prob)
                # 收益率
                q_values = np.zeros(self.n_arms)
                # 统计拉杆次数
                counts = np.zeros(self.n_arms)
                success = np.zeros(self.n_arms)
                failure = np.zeros(self.n_arms)
                
                # 拉1000次杆
                for j in range(self.n_pulls):
                    # 根据不同策略选择臂
                    _arm = self.pick_arm(st, q_values, counts, success, failure)
                    # 选择臂的收益
                    reward = np.random.binomial(1, arm_prob[_arm])
                    # 拉杆次数统计
                    counts[_arm] += 1
                    # 更新收益
                    q_values[_arm] += (reward-q_values[_arm]) / counts[_arm]
                    # 成功的收益
                    success[_arm] += reward
                    # 失败的收益
                    failure[_arm] += (1-reward)
                    best_arm_counts[i][j] = counts[best_arm] * 100.0 / (j+1)
                #print('====== st:{}, i:{} '.format(st, i))
                #print('arm_prob: ', arm_prob)
                
            # 横纵坐标
            y = np.mean(best_arm_counts, axis=0)
            x = range(len(y))
            #print('strategy: ', st)
            ax.plot(x, y, label=st)
            ax.legend(loc=0)
        
        plt.xlabel('steps')
        plt.ylabel('optimal pull')
        plt.tight_layout()
        plt.ylim((0, 110))
        plt.show()

bandits = Bandits()
bandits.plot_result()

总结

random表现最差，采用5个里面随机选择一个杆的方式，选择最高收益杆的概率趋近于0.2。

汤普森采样表现最好，能够最快速的找到收益最大的杆，其次是Epsilon-Greedy、UCB。
